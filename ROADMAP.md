# DeepLocal プロジェクトロードマップ

「DeepLocal」を世界最高のiOS翻訳アプリにするための開発計画です。

## フェーズ 1: 最小実行可能製品 (MVP)
**目標**: `LFM2-350M-ENJP-MT` を使用したオンデバイス翻訳をAppleの `MLX Swift` を用いて実現する。

- [x] 推論エンジンの選定
  - Apple公式の `MLX Swift (mlx-swift-lm)` を採用。
- [ ] モデルの最適化
  - `mlx-community` で公開されている量子化版（4-bit等）の利用検討。
- [ ] 基本UIの実装
  - テキスト入力、翻訳結果表示、ストリーミング出力の対応。
- [ ] 初回起動時のモデル自動ダウンロード機能。

## フェーズ 2: UI/UX の洗練
**目標**: 常用できるレベルまで使い勝手を向上させる。

- [ ] チャット形式の翻訳インターフェース。
- [ ] 翻訳履歴の保存（オンデバイスDB）。
- [ ] お気に入り文言の保存。
- [ ] 音声入力（Speech-to-Text）の統合。
- [ ] オフラインモードの完全対応。

## フェーズ 3: 自由度とカスタマイズ (BYOM: Bring Your Own Model)
**目標**: ユーザーが好きなLLMを翻訳エンジンとして使えるようにする。

- [ ] Hugging Face からのモデルダウンロードUI。
- [ ] 外部ストレージ或いはiCloudからのモデルインポート機能。
- [ ] GGUF / MLX 形式の汎用サポート。
- [ ] モデルごとのシステムプロンプト設定機能。

## フェーズ 4: iOSエコシステムへの統合
**目標**: システム標準の翻訳アプリに近いアクセシビリティを実現する。

- [ ] **Share Extension**: 他のアプリで選択したテキストを直接 DeepLocal で翻訳。
- [ ] **Siri / ショートカット対応**: 声や自動化で翻訳を実行。
- [ ] **標準翻訳機能（予定）**: iOSのシステムワイドな翻訳プロバイダーとしての登録方法の模索。
- [ ] ウィジェット対応。
